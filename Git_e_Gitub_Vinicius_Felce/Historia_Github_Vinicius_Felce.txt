1.Quem inventou o sistema de controle de versão Git e por quê?
Ele foi criado por Linus Torvalds, que é mais conhecido por outra criação, o Linux.
Como na época não existia nada similar, ele resolveu criar a sua própria versão
de um sistema para controlar as alterações do projeto Linux. E foi aí que surgiu o Git.

2.Quem atualmente é o dono do Github e por quê?
A aquisição do GitHub pela Microsoft por US$ 7,5 bilhões em 2018 de junho,
desde que Satya Nadella assumiu a liderança da Microsoft, há mais de quatro anos,
uma parte central de sua estratégia tem sido tentar atrair programadores de código aberto.
