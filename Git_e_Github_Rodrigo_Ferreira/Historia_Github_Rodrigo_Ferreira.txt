1.Quem inventou o sistema de controle de versão Git e por quê?

Resposta:
O Git foi inventado por Linus Torvald, o qual foi o mesmo inventor do kernel Linux. O Git foi inventado porque em 2005, 
ano de sua criação, terminou a gratuidade do serviço proprietário BitKeeper, com o qual Torvald desenvolvia o kernel.
Na época, nenhum outro sistema de comando existente até então atendia as necessidades para que ele continuasse o desenvolvimento.
